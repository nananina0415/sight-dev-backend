# 그룹 멤버 목록 조회

그룹에 참여 중인 멤버 목록을 조회합니다.

## API

```
GET /groups/{groupId}/members
```

### 쿼리 파라미터

(해당 없음)

### 요청 바디

(해당 없음)

### 응답 코드 및 응답 바디

```
200 OK
```

| 이름               |   타입    | 설명                                                     |
| :----------------- | :-------: | :------------------------------------------------------- |
| `members`          | 객체 배열 | 멤버 목록. 그룹장 우선, 이후 `userId` 오름차순 정렬      |
| `members.userId`   |   숫자    | 회원 ID                                                  |
| `members.name`     |  문자열   | 아이디                                                   |
| `members.realname` |  문자열   | 실명                                                     |
| `members.isLeader` |  불리언   | 그룹장 여부 (`group.master == members.userId`인 경우 `true`) |

### 테스트 케이스

1. 열람 권한이 있는 회원이 멤버 목록을 조회할 수 있다
   - `group.master`와 일치하는 멤버의 `isLeader`는 `true`다
   - 그룹장이 목록 맨 앞에 오고, 나머지는 `userId` 오름차순으로 정렬된다

2. 공개 범위(`grade`)에 따라 열람 가능 여부가 결정된다
   - 비공개(`grade=0`): 그룹 멤버만 조회 가능. 비멤버 요청 시 403
   - 운영진 공개(`grade=2`): 운영진만 조회 가능. 일반 회원 요청 시 403
   - 회원 공개(`grade=3`): 재학·휴학·졸업 회원 조회 가능
   - 완전 공개(`grade=4`): 교류 회원 포함 모든 로그인 회원 조회 가능

3. 존재하지 않는 그룹을 조회하면 404를 반환한다
