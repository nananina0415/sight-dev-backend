# 레시피 등록

새로운 요리 레시피를 등록합니다. 레시피 등록 시 재료 목록과 조리 단계를 함께 지정할 수 있습니다.

## API

```
POST /recipes
```

### 쿼리 파라미터

(해당 없음)

### 요청 바디

| 이름                       |   타입    | 설명                          |
| :------------------------- | :-------: | :---------------------------- |
| `title`                    |  문자열   | 레시피 제목                   |
| `servings`                 |   숫자    | 몇 인분인지                   |
| `ingredients`              | 객체 배열 | 재료 목록                     |
| `ingredients.name`         |  문자열   | 재료 이름                     |
| `ingredients.amount`       |  문자열   | 재료 양 (예: "200g", "2큰술") |
| `steps`                    | 객체 배열 | 조리 단계 목록                |
| `steps.order`              |   숫자    | 단계 순서                     |
| `steps.description`        |  문자열   | 단계 설명                     |
| `steps.timer`              |   객체    | 타이머 설정                   |
| `steps.timer.minutes`      |   숫자    | 분                            |
| `steps.timer.label`        |  문자열   | 타이머 라벨 (예: "끓이기")    |

### 응답 코드 및 응답 바디

```
201 Created
```

### 테스트 케이스

1. 레시피를 정상적으로 등록할 수 있다
   - 유효한 요청 바디를 전달하면 201 응답과 함께 생성된 레시피 정보를 반환한다
2. 레시피 제목은 필수이다
   - `title`이 빈 문자열이거나 누락된 경우 400 에러를 반환한다
3. 재료는 최소 한 개 이상이어야 한다
   - `ingredients`가 빈 배열이면 400 에러를 반환한다
4. 조리 단계의 순서는 1부터 시작하는 연속된 숫자여야 한다
   - `steps.order`가 중복되거나 빠진 숫자가 있으면 400 에러를 반환한다
