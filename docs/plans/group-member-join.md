# 그룹 참여

그룹에 참여(가입)합니다.

## API

```
POST /groups/{groupId}/members/@me
```

### 테스트 케이스

1. 조건을 충족한 회원이 그룹에 참여할 수 있다
   - `group_member`에 레코드가 추가되고 `group.count_member`가 1 증가한다
   - `group.changed_at`이 갱신된다. 중단 상태(`suspend`)인 그룹이면 진행 중(`progress`)으로 자동 전환된다
   - 참여자에게 +10 ExPoint가 부여된다. 단, 활용 실습 그룹(`id=7549`)은 ExPoint를 부여하지 않는다
   - 운영 카테고리(`category=manage`) 그룹이면 전 멤버에게 알림이 발송된다. 그 외엔 참여자 본인과 그룹장에게 알림이 발송된다

2. `allow_join`이 꺼진 그룹에는 참여할 수 없다 — 403

3. 그룹 열람 권한이 없으면 참여할 수 없다 — 403

4. 이미 멤버인 경우 400을 반환한다

5. 종료된 그룹에는 참여할 수 없다 — 400

6. 존재하지 않는 그룹이면 404를 반환한다
