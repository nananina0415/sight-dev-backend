# 그룹 멤버 내보내기

그룹장이 특정 멤버를 강제로 내보냅니다.

## API

```
DELETE /groups/{groupId}/members/{memberId}
```

### 쿼리 파라미터

(해당 없음)

### 요청 바디

(해당 없음)

### 응답 코드 및 응답 바디

```
204 No Content
```

### 테스트 케이스

1. 그룹장이 멤버를 정상적으로 내보낼 수 있다
   - `group_member` 레코드가 삭제되고 `group.count_member`가 1 감소한다
   - `group.changed_at`이 갱신된다. 중단 상태(`suspend`)인 그룹이면 진행 중(`progress`)으로 자동 전환된다
   - 내보내진 멤버에게 -10 ExPoint가 차감된다. 활용 실습 그룹(`id=7549`)은 차감하지 않는다
   - 내보내진 멤버에게 알림이 발송된다

2. 그룹장이 아닌 회원이 요청하면 403을 반환한다

3. 운영 카테고리(`category=manage`) 그룹에서는 내보내기를 할 수 없다 — 400

4. 자기 자신을 내보낼 수 없다 — 400

5. 대상이 그룹 멤버가 아니면 400을 반환한다

6. 존재하지 않는 그룹이면 404를 반환한다

7. 존재하지 않는 회원이면 404를 반환한다
