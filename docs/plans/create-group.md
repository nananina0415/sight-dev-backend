# 그룹 생성

새로운 그룹을 생성합니다. 생성자에게 +20 ExPoint가 부여되며, 생성자는 자동으로 그룹장 및 첫 번째 멤버로 등록됩니다.

## API

```
POST /groups
```

### 쿼리 파라미터

(해당 없음)

### 요청 바디

| 이름         |    타입     | 설명                                                                                   |
| :----------- | :---------: | :------------------------------------------------------------------------------------- |
| `title`      |   문자열    | 그룹 이름 (필수)                                                                       |
| `category`   |   문자열    | 카테고리 (필수, `study`, `project`, `documentation`, `education`, `program`, `manage`) |
| `purpose`    |   문자열    | 그룹 목적                                                                              |
| `interest`   | 문자열 배열 | 관심 분야                                                                              |
| `technology` | 문자열 배열 | 기술 스택                                                                              |
| `allowJoin`  |   불리언    | 참여 허용 여부 (기본값: false)                                                         |
| `grade`      |    숫자     | 공개 범위 (0=비공개, 2=운영진, 3=회원, 4=완전공개, 기본값: 3)                          |
| `repository` |   문자열    | 저장소 URL                                                                             |
| `ideaId`     |    숫자     | 아이디어클라우드 아이디어 ID (아이디어 기반 그룹 생성 시)                              |

### 테스트 케이스

1. 그룹을 정상적으로 생성할 수 있다
   - 필수 필드(title, category)를 전달하면 201 응답과 함께 생성된 그룹 정보를 반환한다
   - 생성자가 그룹장(master)이자 첫 번째 멤버로 등록된다
   - `countMember`가 1로 설정된다
2. 그룹 생성 시 생성자에게 +20 ExPoint가 부여된다
   - 생성 후 생성자의 expoint가 20 증가하고 expoint_log에 이력이 기록된다
3. 그룹 이름은 필수이다
   - `title`이 빈 문자열이거나 누락된 경우 400 에러를 반환한다
4. 카테고리는 유효한 값이어야 한다
   - 유효하지 않은 카테고리 값을 전달하면 400 에러를 반환한다
5. 운영 카테고리(`manage`) 그룹은 운영진만 생성할 수 있다
   - 일반 회원이 `manage` 카테고리로 그룹을 생성하려 하면 403 에러를 반환한다
6. 교육 카테고리(`education`) 그룹은 운영진만 생성할 수 있다
   - 일반 회원이 `education` 카테고리로 그룹을 생성하려 하면 403 에러를 반환한다
7. 프로그램 카테고리(`program`) 그룹은 운영진만 생성할 수 있다
   - 일반 회원이 `program` 카테고리로 그룹을 생성하려 하면 403 에러를 반환한다
8. 운영진 공개(grade=2)는 운영진만 설정할 수 있다
   - 일반 회원이 grade를 2로 설정하여 그룹을 생성하려 하면 403 에러를 반환한다
9. 아이디어클라우드 연동으로 그룹을 생성할 수 있다
   - `ideaId`를 전달하면 해당 아이디어의 state가 `expired`로 변경된다
   - 아이디어 작성자에게 알림이 전송된다
10. 존재하지 않거나 이미 만료된 아이디어로는 그룹을 생성할 수 없다
    - 존재하지 않는 `ideaId`를 전달하면 404 에러를 반환한다
    - 이미 `expired` 상태인 아이디어 ID를 전달하면 400 에러를 반환한다
