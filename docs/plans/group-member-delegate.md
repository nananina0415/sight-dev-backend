# 그룹장 위임

현재 그룹장이 다른 멤버에게 그룹장을 위임합니다.

## API

```
PATCH /groups/{groupId}/master
```

### 쿼리 파라미터

(해당 없음)

### 요청 바디

| 이름       |  타입  | 설명                      |
| :--------- | :----: | :------------------------ |
| `memberId` |  숫자  | 새 그룹장으로 위임할 회원 ID |

### 응답 코드 및 응답 바디

```
204 No Content
```

### 테스트 케이스

1. 그룹장이 다른 멤버에게 그룹장을 위임할 수 있다
   - `group.master`가 위임 대상 회원 ID로 업데이트된다
   - `group.changed_at` 갱신 여부는 기존 구현 확인 필요 (결정 보류)
   - 전 멤버에게 알림이 발송된다

2. 그룹장이 아닌 회원이 요청하면 403을 반환한다

3. 위임 대상이 그룹 멤버가 아니면 400을 반환한다

4. 자기 자신에게 위임하면 400을 반환한다

5. 존재하지 않는 그룹이면 404를 반환한다

6. 존재하지 않는 회원이면 404를 반환한다
